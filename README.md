# 주요 기능
- 일반
  - 회원가입 / 로그인
  - 캐릭터 생성 / 선택
  - 미니맵
  - 캐릭터, NPC, 맵 그래픽
  - 메시지
  - 인벤토리 아이템 장착/사용
  - 상점 구입/판매
  - 장비창 장비 해제
  - 상태창
  - 퀘스트 진행 현황

- 스레드
  - 배경음악 스레드 (영상에는 포함되어있지 않습니다.)
  - 회복 스레드
  - 몬스터 공격 스레드
  - 캐릭터 공격 스레드
  - 스킬 스레드
  - 자동 길찾기 스레드
  - 게임 팁 스레드

# 스택
- Java 11
- 자바 쓰레드
